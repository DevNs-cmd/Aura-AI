# Aura AI

Aura AI is a multi-component system comprising a backend API service, a frontend client, and database components (schema/migrations/seed). This repository is structured as a monorepo to keep development and operations consistent across services.

## Repository layout

- `backend/` – Backend API service (Node.js/NestJS)
- `frontend/` – Frontend application (Flutter)
- `database/` – Database assets (schema, migrations, seed, scripts)
- `docs/` – Architecture, API, database, deployment, and other documentation
- `scripts/` – Developer/ops scripts
- `docker/` – Containerization assets

## Documentation

See `docs/` for architecture notes, API documentation, and deployment guidance.
