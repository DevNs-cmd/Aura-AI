# Aura AI - Premium Companion App

Aura AI is a production-ready, premium AI Companion Flutter application (similar to ChatGPT, Gemini, and Apple Intelligence) built with pixel-perfect replication of provided designs.

## Features Built

- **Splash Screen**: Breath animations and automatic redirection logic.
- **Authentication**: Sign In & Register views, custom validators, password toggles, Google/Apple buttons.
- **Home Dashboard**: Quick actions, mindfulness logs, mood selector, and recent memory logs.
- **Chat**: monospaced code blocks with copy utilities, typing indicators, and call triggers.
- **Journal**: Emoji mood scores, AI writing prompts, diary compositions, and entries.
- **Goals**: Weekly Bezier lines graph charts and donut segmented focus indicators built programmatically.
- **Long-Term Memory**: Fact databases, category filters, unread badges, and pin-toggle states.
- **Voice Assistant**: Dark calling overlay theme, breathing morphing orb vectors, and multi-sine wave audio frequencies.
- **Vision Analysis**: Camera/Gallery capture mock uploads, glowing laser scanner line custom painters, detected objects progress bars, and Extracted OCR cards.
- **Profile**: STATS grid rows, personality tonality selection grids, and usage insights custom vertical bar charts.
- **Settings**: Grouped modern card layouts, custom SwitchListTile themes, language/pitch selection checkmark dialogs, cache managers, and red logout actions.
- **Notifications**: Reminders list grouped into Today vs Earlier, priority colored pills, unread dot badges on bell icon and cards, and dismiss options.

## Getting Started

### Prerequisites
- **Flutter SDK**: Ensure you have Flutter SDK installed (v3.35.0+ recommended).
- Run `flutter doctor` to verify your environment setup.

### Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/khelseekho-ui/aura-ai.git
   cd aura-ai
   ```

2. **Get Dependencies**:
   ```bash
   flutter pub get
   ```

### Running the Application

To run the application on a connected device, simulator, or browser:
```bash
flutter run
```

To target a specific platform:
- **Web (Chrome)**:
  ```bash
  flutter run -d chrome
  ```
- **Windows Desktop**:
  ```bash
  flutter run -d windows
  ```

### Running Tests

Run the comprehensive unit test suite to verify state controllers:
```bash
flutter test
```
